@name not
@title Not
@summary Logically invert an expression

@description 

If `value` is `true` then return `false`, and if `false` then return `true`. The exclamation operator `!` is an alias for `not` e.g. `!(x>0)` is equivalent to `not(x>0)`.

@param {boolean} value The value to be negated
@return {boolean} The logical inverse

@example not(x < 4)
